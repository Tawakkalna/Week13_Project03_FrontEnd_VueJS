## How To Use

From your command line:

#### # Clone this repository

```bash
git clone https://github.com/Tawakkalna/vue.git
```

#### # Go into the frontend directory

```bash
cd vue/twakkalna
```

#### # Install the dependencies

```bash
yarn install
```

#### # Run the app

```bash
npm start
```
